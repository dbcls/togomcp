TogoMCP

A comprehensive Model Context Protocol (MCP) server providing LLM agents with seamless access to over 20 major life sciences databases through SPARQL queries, RDF data exploration, and intelligent ID conversion services.

MCP Endpoint:
https://togomcp.rdfportal.org/mcp

What is TogoMCP?

TogoMCP is a powerful bridge between AI assistants and the vast ecosystem of biological and biomedical databases. Developed by the Database Center for Life Science (DBCLS), it enables researchers to interact with complex life sciences data using natural language through AI assistants like Claude, ChatGPT, and other MCP-compatible clients.

Key Features:

  • Unified Access: Query 20+ databases including UniProt, PubChem, ChEMBL, PDB, Reactome, and more through a single interface
  • Intelligent Search: Specialized search tools for proteins, chemicals, pathways, and diseases
  • SPARQL Integration: Direct access to RDF databases with example queries and best practices
  • ID Conversion: Seamlessly convert between different database identifiers (TogoID integration)
  • Expert Guidance: Built-in MIE (Metadata Interoperability Exchange) files guide optimal query construction

Endpoint: https://togomcp.rdfportal.org/mcp

Usage Examples

Example 1: Cardiovascular Disease Protein-Drug Analysis

Prompt: Find proteins that are associated with both cardiovascular diseases and have known small molecule inhibitors in ChEMBL, and classify them according to disease and drug availability.

Response Process:

The AI assistant systematically identified 75+ human proteins associated with cardiovascular diseases with known inhibitors by:

  1. Searching MeSH for cardiovascular disease terms
  2. Querying ChEMBL for cardiovascular targets using specialized search tools
  3. Constructing SPARQL queries to retrieve detailed protein-inhibitor relationships
  4. Analyzing drug development status and therapeutic classes

Tools Used:

  • list_databases - Identified relevant databases
  • search_mesh_entity - Found cardiovascular disease terms
  • search_chembl_target - Searched for cardiovascular targets
  • get_MIE_file - Retrieved ChEMBL schema and examples
  • run_sparql - Executed multiple queries for comprehensive data

Key Results:

  • Identified 15+ targets for hypertension with approved drugs
  • Found 20+ targets for coronary artery disease (approved + developmental)
  • Discovered 57 sodium channel inhibitors and 27 calcium channel blockers
  • Categorized targets by disease, drug status, and therapeutic class

Example 2: Human Enzyme Distribution Analysis

Prompt: Determine the distribution of human enzymes by their EC number classes.

Response Process:

The analysis categorized 4,442 unique human enzymes from UniProt's Swiss-Prot database across seven EC classes, revealing that transferases (41%) and hydrolases (37.3%) dominate human enzyme repertoire.

Tools Used:

  • get_MIE_file - Retrieved UniProt schema and SPARQL examples
  • run_sparql - Queried enzyme data with EC classifications

Key Results:

  • EC 2 (Transferases): 1,823 proteins (41.0%)
  • EC 3 (Hydrolases): 1,655 proteins (37.3%)
  • EC 1 (Oxidoreductases): 545 proteins (12.3%)
  • Smaller classes (EC 4-7) represent specialized metabolic functions (2.3-3.4% each)

Example 3: Natural Products vs Synthetic Compounds

Prompt: Use ChEBI (chemical classification) + ChEMBL (bioactivity) + PubChem (chemical descriptors) to compare the bioactivity profiles and chemical diversity of natural products versus synthetic compounds across different therapeutic areas.

Response Process:

A comprehensive multi-database analysis comparing natural products (e.g., Vinblastine, Paclitaxel) with synthetic compounds (e.g., Imatinib, Omeprazole) revealed distinct patterns in potency, selectivity, and chemical space occupation.

Tools Used:

  • get_MIE_file - Retrieved schemas for ChEBI, ChEMBL, PubChem
  • run_sparql - Queried chemical classifications and bioactivity data
  • get_compound_attributes_from_pubchem - Retrieved molecular descriptors

Key Results:

  • Natural Products: 70% show IC50 < 100 nM, average 8 chiral centers, hit 8.5 targets (polypharmacology), dominant in oncology
  • Synthetic Compounds: 40% show IC50 < 100 nM, average 1 chiral center, hit 2-3 targets (selective), dominant in cardiovascular/CNS/metabolic diseases
  • Natural products occupy unique 3D chemical space with higher sp3 fraction (0.6 vs 0.3)
  • Complementary strengths suggest optimal strategy combines both approaches

Setup Guide

Claude Desktop

TogoMCP is available for Claude users on Pro, Max, Team, and Enterprise plans.

Method 1: Using the Settings UI (Recommended)

Navigate to Settings → Connectors
Open Claude Desktop or Claude.ai and access the Connectors section in Settings.
Click "Add custom connector"
Scroll to the bottom of the Connectors section and click the "Add custom connector" button.
Enter the TogoMCP URL
Add the server URL: https://togomcp.rdfportal.org/mcp
Configure and Add
Click "Add" to finish the configuration. TogoMCP is ready to use!
Enable in Conversation
Click the "Search and tools" button in the chat interface and enable TogoMCP for your conversation.

Method 2: Editing Configuration File

For advanced users who prefer to edit the configuration file directly:

Locate the configuration file
Find your Claude Desktop configuration file:
  • macOS: ~/Library/Application Support/Claude/claude_desktop_config.json
  • Windows: %APPDATA%\Claude\claude_desktop_config.json
Edit the configuration file
Open the file in a text editor and add TogoMCP using the mcp-remote wrapper:
{ "mcpServers": { "togomcp": { "command": "npx", "args": [ "mcp-remote", "https://togomcp.rdfportal.org/mcp" ] } } }
Save and restart Claude Desktop
Save the configuration file and restart Claude Desktop for the changes to take effect.
Verify installation
Click the "Search and tools" button in the chat interface. You should see "togomcp" listed among available tools.

Note: If you have multiple MCP servers, add them as separate entries within the mcpServers object. The mcp-remote package enables Claude Desktop to connect to remote HTTP/HTTPS MCP servers.

ChatGPT

TogoMCP is available for ChatGPT Business and Enterprise/Edu customers through Developer Mode.

Enable Developer Mode
Go to Settings → Connectors → Advanced, and toggle on "Developer Mode".
Add Custom Connector
Navigate to Settings → Connectors and click "Create" or "Add custom connector".
Configure TogoMCP
Enter the connector details:
  • Name: TogoMCP
  • URL: https://togomcp.rdfportal.org/mcp
  • Description: Life sciences database integration via MCP
Create and Authorize
Click "Create" and authorize ChatGPT to use the connector.
Use in Chat
In the chat panel, enable "Developer Mode" and select TogoMCP from available tools.

Gemini CLI

Note: TogoMCP is provided via Streamable-HTTP, not SSE.

Create settings.json
Create or edit your Gemini CLI settings file with the following configuration:
{ "mcpServers": { "togomcp": { "httpUrl": "https://togomcp.rdfportal.org/mcp" } } }
Save and Restart
Save the settings file and restart Gemini CLI to activate TogoMCP.

Available Databases

TogoMCP provides access to 20+ curated life sciences databases:

UniProt

Comprehensive protein sequences and functional information with 444M proteins. Includes Swiss-Prot (manually curated) and TrEMBL (automatically annotated).

PubChem

119M chemical compounds with molecular descriptors, 339M substances, 1.7M bioassays, and extensive bioactivity data.

ChEMBL

2.4M+ bioactive molecules with drug-like properties, 1.6M assays, and 20M bioactivity measurements.

PDB (Protein Data Bank)

204K+ 3D structural data for proteins, nucleic acids, and complexes from X-ray, NMR, and cryo-EM experiments.

Reactome

22,000+ curated biological pathways and processes across 30+ species with molecular interactions and disease associations.

Rhea

17,078 expert-curated biochemical reactions, all atom-balanced and linked to ChEBI, enzyme classifications, and pathway databases.

ChEBI

217,000+ chemical entities of biological interest with hierarchical classification, molecular data, and biological roles.

Gene Ontology (GO)

Controlled vocabulary for gene and gene product attributes: 30,804 biological processes, 12,793 molecular functions, 4,568 cellular components.

NCBI Gene

57M+ gene entries covering protein-coding genes, ncRNAs, and pseudogenes across all organisms with orthology relationships.

Ensembl

Genome annotations for 100+ species with genes, transcripts, proteins, and cross-references to UniProt, HGNC, and OMIM.

MONDO

Comprehensive disease ontology integrating OMIM, Orphanet, DOID, MESH, ICD, and 35+ databases for precision medicine.

MeSH

NLM's controlled vocabulary thesaurus for biomedical literature with hierarchical descriptors and qualifiers across 16 main categories.

ClinVar

3.5M+ variant records with clinical interpretations, gene associations, and disease conditions for genomic medicine.

MedGen

233,000+ clinical concepts covering diseases, phenotypes, and clinical findings with links to OMIM, Orphanet, and HPO.

PubMed

Bibliographic information for biomedical literature from MEDLINE with MeSH annotations and cross-references.

PubTator Central

Biomedical entity annotations from PubMed literature using text mining and manual curation.

NCBI Taxonomy

Comprehensive taxonomic classification covering 3M+ organisms with hierarchical relationships and genetic code assignments.

BacDive

97,000+ bacterial and archaeal strain records with phenotypic and genotypic characterizations.

MediaDive

3,289 standardized culture media recipes for bacteria, archaea, fungi, yeast, microalgae, and phages.

GlyCosmos

Comprehensive glycoscience portal integrating glycan structures, glycoproteins, and lectin-glycan interactions.

DDBJ

Nucleotide sequence data from INSDC with genomic annotations and cross-references to BioProject and BioSample.

NANDO

2,777 Japanese intractable (rare) disease classes with multilingual labels and cross-references to MONDO.

Available Tools

TogoMCP provides specialized tools for efficient database interaction:

Database Management
  • list_databases: Get all available databases with descriptions
  • get_sparql_endpoints: Retrieve SPARQL endpoint URLs
  • get_graph_list: List named graphs in a database
Schema & Examples
  • get_MIE_file: Retrieve Metadata Interoperability Exchange (MIE) files with schemas, RDF samples, and SPARQL examples
  • get_sparql_example: Get example SPARQL queries for a database
  • keyword_search_instructions: Get search guidance for specific databases
Entity Search - Proteins
  • search_uniprot_entity: Search UniProt proteins by keywords
  • search_pdb_entity: Search PDB structures (pdb/cc/prd databases)
Entity Search - Chemicals
  • search_chembl_molecule: Search ChEMBL small molecules
  • search_chembl_target: Search ChEMBL drug targets
  • search_chembl_id_lookup: Lookup ChEMBL IDs
  • get_pubchem_compound_id: Get PubChem CID from compound name
  • get_compound_attributes_from_pubchem: Retrieve compound attributes
Entity Search - Pathways & Reactions
  • search_reactome_entity: Search Reactome pathways, reactions, and complexes
  • search_rhea_entity: Search biochemical reactions in Rhea
Entity Search - Diseases
  • search_mesh_entity: Search MeSH medical concepts
SPARQL Queries
  • run_sparql: Execute SPARQL queries on any database
ID Conversion (TogoID)
  • togoid_convertId: Convert IDs between databases
  • togoid_countId: Count convertible IDs
  • togoid_getAllRelation: Get all conversion relationships
  • togoid_getDataset: Get dataset configuration
  • togoid_getDescription: Get database descriptions
Usage Guidance
  • TogoMCP_Usage_Guide: Comprehensive workflow guide for optimal tool usage

Related Resources

TogoMCP Server

URL: https://togomcp.rdfportal.org/mcp

The main TogoMCP endpoint providing MCP access to life sciences databases. Developed and maintained by DBCLS to enable seamless integration between AI assistants and biological databases.

RDF Portal

URL: https://rdfportal.org

A comprehensive repository for semantic data in life sciences (formerly NBDC RDF Portal). Provides SPARQL endpoints, dataset downloads, and unified access to Japanese life science RDF databases. The portal houses the underlying infrastructure that TogoMCP builds upon, offering direct SPARQL access to 20+ curated databases with over 45 billion triples.

TogoID

URL: https://togoid.dbcls.jp

An ID conversion and exploration service that enables seamless navigation between different biological databases. TogoID powers the ID conversion tools in TogoMCP, allowing researchers to transform identifiers from one database format to another (e.g., UniProt to PDB, PubChem to ChEBI).

DBCLS (Database Center for Life Science)

URL: https://dbcls.rois.jp

The Database Center for Life Science, part of the Research Organization of Information and Systems (ROIS), develops and maintains integrated databases and tools for life science research. DBCLS creates TogoMCP, RDF Portal, TogoID, and numerous other resources to advance data integration and accessibility in biological research through semantic web technologies and standardized protocols.
